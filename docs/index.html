  • The Inweb Manual and Source Code

§1. This is the home page for documentation, human-readable source code and examples for the literate programming system Inweb. It's hosted on the Github Docs area of the Inweb repository, at: https://github.com/ganelson/inweb Download and build instructions are there, not here.

To get a sense of Inweb and what it does, see inweb/M-iti.html. The simple example twinprimes/twinprimes.html and goldbach/index.html may also help.

But if you're here to look under the hood, or you want to fix a bug or add a feature, a good place to begin is inweb/P-htpw.html.

§2. Inweb is built on top of a library of C functions called foundation-module/index.html, supplied in the same repository. See foundation-module/P-abgtf.html. There's also a small tool for unit-testing it, foundation-test/index.html, but this is not an interesting read.
